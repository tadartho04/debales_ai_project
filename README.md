# AI Chatbot with RAG + Routing

## Features
- RAG-based answers for Debales AI queries
- General query handling (SERP simulation)
- LangGraph-based routing

## Tech Used
- Python
- LangChain
- LangGraph
- FAISS

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python app.py

## Example Queries
- What is Debales AI?
- What is Machine Learning?

## Note
Used a simplified embedding approach (FakeEmbeddings) instead of paid APIs due to resource constraints.
